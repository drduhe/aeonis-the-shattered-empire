This packet defines what turns **on** beyond `First_Playable_Packet.md` once First Playable is validated (see the exit gate in `Balance_Dashboard.md` §6). It is the target scope for the retail box and the 4-10 hour session experience (locked decision D1, `../marketing/Positioning.md`).

## Sequencing note (why each hour stays interesting)

The full game is tuned so something new comes online roughly every stage of play:

1. **Rounds 1-2:** expansion, Tier I research, first council laws.
2. **Rounds 3-4:** first artifacts (Remnant target), Tier II unlocks, Elite units (Forge gate), Accords form.
3. **Rounds 5-6:** Legendary Buildings, Advanced units (Tier II gate), Stage II objectives, Seat contention.
4. **Rounds 7+:** Tier III capstones, Mythic units (Legendary gate), Elections and endgame agendas, betrayals.

If playtests show a sagging stage, reach for the matching dial in `Balance_Dashboard.md` §5.

## Playtest goals specific to full scope

1. Do Tier II/III discoveries create distinct tech identities (per school) or does everyone convergent-draft the same picks?
2. Do faction-unit unlock gates (Forge / Tier II / Legendary) pace the military escalation correctly?
3. Do Accords form early and break late (healthy) or never form / never break?
4. Do Stage II objectives redirect the mid-game, TI4-style?
5. Do Elections create table drama without kingmaking complaints?
6. Session length: does 4-6p land inside 4-10 hours (D1)? Where does time go (log per-phase minutes once)?
