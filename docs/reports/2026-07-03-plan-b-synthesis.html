3 · Three design calls

Call 1 — Plan 1 first. Combat is a real VP path (H5 killed). The original Plan 1 problem statement assumed rational players don't attack; persona data refutes that at bot skill level. Run the A/B ladder: baseline → Aggressor's Edge → Pillage. Re-run 100-game Warmonger bracket after each step.
Call 2 — Plan 3 is the structural fix. Seat dominance persists (H1) and objectives don't carry wins (H3). Plan 3's Coronation Rite / shared objectives / cut purchased VP is validated as necessary — not optional polish. Scope an MVP slice before full doc sweep.
Call 3 — Defer 7p engine work. H4 killed. High-count validation shifts to group-play feel (Plan 4), not sim pacing. 8p shows Seat share dropping to 46% — player dilution helps naturally.
